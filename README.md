# toolsGM 

## #pl-PL

### Opis:
Jest narzędziem dla graczy RPG, 
jest to wirtualny zestaw narzędzi takich jak:
- Plansza do gry
- Archiwum grafiki (map i fotografii)
- Odtwarzacz muzyki
- Odtwarzacz dźwięków tła
- Notatki i karteczki
- Zegarek
- Minutnik
- Chat

### Technologie
- React.js
- Drag&drop
- Player audio
- CSS 3D
- PWA
- GIT

### Urchomianie

Potrzebujesz zainstalować node.js

```bash
npm install
```

```bash
npm run start
```
